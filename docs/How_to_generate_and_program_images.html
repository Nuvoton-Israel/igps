1. Image Creation

Image Creation Files Description

.\

Generating_Images.bat                           Generates the output_binaries out of the inputs files

UpdateInputsBinaries.bat                        copies Poleg_bootblock.bin and u-boot.bin from existing versions  user can edit and run it before running Generating_Images.bat in order to copy different versions or debug versions

 

.\inputs  the input binary files and their descriptions + the security keys

Poleg_bootblock.bin                                 BootBlock binary (body)  user can copy instead a new version (see UpdateInputsBinaries.bat)

BootBlockAndHeader.xml                       BootBlock Header description in Bingo XML format  user can edit its fields

u-boot.bin                                                   u-boot binary (body)  you can copy instead a new version (see UpdateInputsBinaries.bat)

UbootHeader.xml                                      u-boot Header description in Bingo XML format  user can edit its fields

mergedBootBlockAndUboot.xml           describes how to concatenate BootBlock and u-boot into one binary image in Bingo XML format

poleg_key_map.xml                                  Poleg Key array (fuse) description in Bingo XML format  user can edit it to add AES keys

poleg_fuse_map.xml                                Poleg Fuse array description in Bingo XML format  user can edit its fields according to its security policy

rsa_private_key_0.bin                              Private key 0 to sign with the images

rsa_public_key_0.bin                                Public key 0 to put into key 0 fuses according to poleg_key_map.xml

rsa_private_key_1.bin                              Private key 1 to sign with the images (when key 0 is invalid)

rsa_public_key_1.bin                                Public key 1 to put into key 1 fuses according to poleg_key_map.xml

rsa_private_key_2.bin                              Private key 2 to sign with the images (when key 0 and 1 are invalid)

rsa_public_key_2.bin                                Public key 2 to put into key 2 fuses according to poleg_key_map.xml and poleg_fuse_map.xml

 

.\output_binaries

        Basic  the generated binaries for basic boot

BootBlockAndHeader.bin                        Bootblock and its header to be programed to SPI, Generated from BootBlockAndHeader.xml

UbootAndHeader.bin                               u-boot and its header to be programed to SPI, Generated from UbootHeader.xml

mergedBootBlockAndUboot.bin            Merged BootBlock and u-boot and their headers to be programed to SPI, Generated from mergedBootBlockAndUboot.xml

        Secure  the generated binaries for secure boot

BootBlockAndHeader.bin                        Bootblock and its signed header to be programed to SPI, Generated from BootBlockAndHeader.xml and keys

UbootAndHeader.bin                               u-boot and its signed header to be programed to SPI, Generated from UbootHeader.xml and keys

mergedBootBlockAndUboot.bin            Merged BootBlock and u-boot and their signed headers to be programed to SPI, Generated from mergedBootBlockAndUboot.xml

poleg_fuse_map.bin                                 Fuse array to be programmed to Fuse array, Generated from poleg_fuse_map.xml

poleg_key_map.bin                                   Key array to be programmed to Key array fuses, Generated from poleg_key_map.xml

 

.\tools  set of tools for generating images

bingo.exe                                                    Binary Generator Generate a Binary from XML with ECC support (see docs\BinaryGenerator_Bingo.doc)

signIt.exe                                                     Command-line tool which can produce RSA-SHA signature for binary images, and embed the signature in the given file  (see docs\SignIt_Readme.txt)

openssl.exe                                                 Open SSL tool used by signIt.exe

 

.\docs  tools documentations

 

.\versions  holds latest and/or old versions to be loaded into inputs directory

Generate Images as is

·       Run Generating_Images.bat to generate the output_binaries out of the inputs files

Modifying and Generating the images

1.      Update inputs\Poleg_bootblock.bin and inputs\u-boot.bin to the up-to date version, you can do it by editing UpdateInputsBinaries.bat

2.      Update the RSA keys files in inputs directory to your own generated keys.

3.      Update the optional fields in inputs\BootBlockAndHeader.xml and in inputs\UbootHeader.xml

4.      Update inputs\poleg_key_map.xml if you want to add AES keys to it

5.      Update inputs\poleg_fuse_map.xml if you want to change your security policy

6.      Run Generating_Images.bat

 

2. Image Programming

Programming Files Description

.\

UART_env.bat                                            Sets the default UART port and baud-rate, if they are wrong it will search for the correct ones

ProgramBootblock_Basic.bat                  Program output_binaries\Basic\BootBlockAndHeader.bin (The BootBlock and its header in Basic mode)

ProgramBootblock_Secure.bat               Program output_binaries\Secure\BootBlockAndHeader.bin (The BootBlock and its header in Secure mode)

ProgramAll_Basic.bat                               Program output_binaries\Basic\mergedBootBlockAndUboot.bin (The BootBlock + u-boot and their header in Basic mode)

ProgramAll_Secure.bat                            Program output_binaries\Secure\mergedBootBlockAndUboot.bin (The BootBlock + u-boot and their header in Secure mode)
also program poleg_key_map.bin and poleg_fuse_map.bin from output_binaries\Secure into their fuse arrays

 

.\programming_tools  set of tools and binaries to help program the

 

.\tools  set of tools for generating images

bingo.exe                                                    Binary Generator Generate a Binary from XML with ECC support (see docs\BinaryGenerator_Bingo.doc)

How to Program Images

One time setup:

1.     Generate the output_binaries as described in Image Creation Files Description

2.     (optional) Update UART_env.bat which sets the default UART port and baud-rate

Programming:

1.     Connect a serial port (via COM port or USB to Serial) to Serial Interface 2

2.     Set strap9 to active low and issue power-on reset

3.     Run one of the scripts described in Programming Files Description
Note that running ProgramAll_Secure.bat which program also the fuses is irreversible.
